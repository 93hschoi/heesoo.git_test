github 사용법입니다.

git에 접속
: git init

git 저장소에 연결
: git remote add origin 'repository address'

git 저장소로 자료 올리기
: git push origin master

git 저장소에서 자료 가져오기
: git pull origin master

git 싱크상태 연결체크
: git status

local folder의 변경사항들을 모두 tracking!
: git add . 

git변경사항에 주석달기
: git commit -m"message"

<push과정>
git status
git add . 
git commit -m "message"
git push origin master

<pull과정>
git pull origin master
