github 사용법입니다.

git에 접속
: git init

git 저장소에 연결
: git remote add origin 'repository address'

git 저장소로 자료 올리기
: git push origin master

git 저장소에서 자료 가져오기
: git pull origin master

git 싱크상태 연결체크
: git status

local folder의 변경사항들을 모두 tracking!
: git add . 

git변경사항에 주석달기
: git commit -m"message"

git 연결 상태확인
: git remote -v

<push과정>
git init
git status
git add . 
git commit -m "message"
git push origin master

<pull과정>
git init
git remote add origin '가져올 git주소'
git pull origin master


* 항상 저장소에서 변경된 파일들이 있다면 그 파일들을 모두 pull한후에 내자료들을
  push가능
* push전엔 pull이 있다!!!
* 저장소에서 변경가능, 꼭 pull 해놓기!
